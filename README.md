# 🔮 전생 알아보기

이름을 입력하면 AI가 당신의 전생을 알려주는 흥미유발형 웹 서비스입니다.
기원전 15만년 전부터 1980년까지, 인간부터 공룡·미생물·외계인까지 — 예측 불가능한 전생 스토리를 만나보세요.

---

## 📌 서비스 구성

```
deploy-test2/
├── server.js                 # Express 서버 + OpenAI API 연동
├── public/
│   └── index.html            # 프론트엔드 (단일 페이지)
├── data/
│   ├── occupations.json      # 300개 직업/존재 데이터
│   └── deaths.json           # 100개 사망 원인 데이터
├── .env                      # OpenAI API Key (Git 미포함)
├── .gitignore
└── package.json
```

### 핵심 흐름

```
사용자 이름 입력
    ↓
랜덤 시대/년도 배정 (역사 시대에 맞는 연도 자동 매핑)
    ↓
랜덤 직업/존재 배정 (300개 중 1개)
    ↓
랜덤 사망 원인 배정 (100개 중 1개)
    ↓
OpenAI API (gpt-4o-mini)로 스토리텔링 생성
    ↓
결과 표시 + 공유하기
```

## 🚀 사용법

### 1. 사전 준비

- [Node.js](https://nodejs.org/) 설치 (v18 이상 권장)
- [OpenAI API Key](https://platform.openai.com/api-keys) 발급

### 2. 설치 및 실행

```bash
# 레포 클론
git clone git@github.com:yyunique89-jpg/deploy-test2.git
cd deploy-test2

# 패키지 설치
npm install

# .env 파일 생성 후 API 키 입력
echo "OPENAI_API_KEY=sk-your-key-here" > .env

# 서버 실행
npm start
```

### 3. 접속

브라우저에서 `http://localhost:3000` 접속 → 이름 입력 → 전생 결과 확인

## 📤 공유 기능

결과 화면에서 **공유하기** 버튼을 누르면:

- **카카오톡** — 카카오스토리로 공유
- **X (트위터)** — 트윗으로 공유
- **텍스트 복사** — 클립보드에 결과 복사 (메신저, SNS 등에 붙여넣기)

## 💡 응용 예시

### 1. 소셜 바이럴 콘텐츠
이름만 입력하면 되는 간단한 구조로 SNS에서 "내 전생 결과" 공유 유도.
친구들끼리 결과를 비교하며 자연스러운 바이럴 효과.

### 2. 이벤트/축제 부스 활용
오프라인 행사에서 태블릿을 세워두고 방문자들이 전생을 확인하는 체험형 부스로 활용.

### 3. 커플/친구 궁합 확장
두 사람의 전생 결과를 비교하여 "전생 궁합" 기능 추가 가능.

### 4. 오늘의 운세 연동
전생 직업을 기반으로 현생 운세를 추가 생성하는 기능으로 확장.

### 5. 교육용 콘텐츠
전생 시대 정보에 실제 역사 설명을 추가하여, 재미와 학습을 겸한 에듀테인먼트 서비스로 발전.

### 6. 다국어 확장
프롬프트와 UI를 영어/일본어 등으로 확장하면 글로벌 서비스로 전환 가능.
